## IG Downloader app

- Instagram Reel Downloader is an Android App. 
- This is build using Jetpack Compose framework.

### How to use?

- Copy Instagram reel link.
- Paste into this app.
- Click on **Get** button.
- Wait few seconds.
- Click on **Download Now** button. 

### Software Requirement

- Android Studio
- Java JDK - 17
- Gradle 8.7
- Gradle Plugin 8.5.2
